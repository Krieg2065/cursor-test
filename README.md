## Getting Started

### Prerequisites

- Git installed on your local machine
- Your favorite code editor (we recommend Cursor!)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Krieg2065/cursor-test.git
```

2. Navigate to the project directory:
```bash
cd cursor-test
```
